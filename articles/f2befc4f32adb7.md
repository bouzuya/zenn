Claude Code を起動するとマスコットキャラクターが表示されます。

![](https://github.com/user-attachments/assets/371f50aa-a2c4-40d7-bb55-ea862402e501)

かわいい？ですね。

このキャラクターの名前が気になったので調べてみたのですが、公式のドキュメントでの説明を見つけられませんでした。

どうもドキュメントに追加しようとする Issue もあるようなので、おそらく記載されていないのでしょう。

<https://github.com/anthropics/claude-code/issues/8536>

この Issue によると公式の X (Twitter) 投稿で **Clawd** と呼ばれていると書かれています。

<https://x.com/trq212/status/1972784970054893877>

どうやら Claude Code を起動すると表示されるマスコットキャラクターの名前は「 Clawd 」のようです。
